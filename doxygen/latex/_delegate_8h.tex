\doxysubsection{Detailed Description}
Delegate series of classes are used to invoke a function synchronously. 

Delgates support binding to free functions, class instance functions, class static function, and std\+::function targets. Lambda functions can be bound to a delegate when assigned to a {\ttfamily std\+::function}. The classes are not thread safe.
