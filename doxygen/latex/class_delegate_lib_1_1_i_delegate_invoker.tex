\doxysubsection{Detailed Description}
Abstract base class to support asynchronous delegate function invoke on destination thread of control. 

Inherit form this class and implement \doxylink{class_delegate_lib_1_1_i_delegate_invoker_a1386e5a1c2650fb1e5511a16cbb5b1f6}{Delegate\+Invoke()}. The implementation typically posts a message into the destination thread message queue. The destination thread receives the message and invokes the target bound function.
