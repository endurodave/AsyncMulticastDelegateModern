Detailed Description

Helper functions for creating dynamic copies of function arguments stored within a tuple.

The template function make_tuple_heap() creates dynamic copies of function arguments, storing them within a tuple for transport through a thread message queue. It supports all types of function arguments, including by value, pointer, pointer-to-pointer, and reference.

The destination thread uses std::apply() to invoke the target function using the tuple of arguments. See Invoke() and DelegateAsyncMsg() in the file DelegateAsync.h for example usage.
