Detailed Description

Abstract base class to support asynchronous delegate function invoke on destination thread of control.

Inherit form this class and implement Invoke(). The implementation typically posts a message into the destination thread message queue. The destination thread receives the message and invokes the target bound function. Destination thread invoke example:

// Get pointer to DelegateMsg data from queue msg data
auto delegateMsg = msg->GetData();

// Invoke the delegate destination target function
delegateMsg->GetDelegateInvoker()->Invoke(delegateMsg);
